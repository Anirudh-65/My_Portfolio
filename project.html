DSP Simulator and Trainer

My project in the course - "ES116: Principles and Applications of Electrical Engineering" involved creating a Digital Signal Processing (DSP) Simulator. This project is based on the Arduino platform and aims to help learners understand the complex field of DSP.

By using this project, learners can directly explore and experiment with DSP algorithms, such as filters, echoes, and Fourier transforms, while working with real-time audio signals. This project combines theoretical knowledge with practical application, creating an engaging learning experience.
